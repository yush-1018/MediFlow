# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in MediFlow, please do not open a public
GitHub issue.

Instead, contact the maintainers privately by email or
[Discord](https://discord.gg/B4Z8MKmzcz) with:

- A description of the issue
- Steps to reproduce
- Potential impact
- Any suggested mitigation (optional)

We will acknowledge your report as soon as possible and work to resolve the issue
before public disclosure.

We appreciate responsible disclosure and thank you for helping keep MediFlow
secure.
